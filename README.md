# Student Directory

Activity for CC400/CS106 - Application Development and Emerging Technologies

By: Prince Venmar T. Bolo - BSIT

A simple React app where you can register students, view the list, delete a student, and see the total number of registered students.

## Features
- Registration form with validation
- Add student
- Display students
- Delete student
- Student counter

## How to run
npm install
npm run dev
